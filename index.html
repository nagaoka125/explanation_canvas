canvasAPIの一部機能の解説

概要

本ページでは、canvasについてその概要とjavascriptでの一部の基礎的な使用方法，実際に実行した様子を示し，
canvasの直線描画, 円描画, テキスト描画, 移動・回転, アニメーションの基礎知識を知ることができます．

canvas

canvasはHTML5で導入された描画要素で，JavaScriptなどを使用してグラフィックを描画する領域を作ることができます．
canvasでは図形を描画するだけでなく，画像の読み込みやアニメーションを作製することもできます．
canvasの座標は左上隅が(0,0)で, X軸が右方向, Y軸が下方向に伸びるようになっています.

canvasの記述方法

HTMLで直接コードを書き込む場合は以下のコードのように記述します．

Java Scriptファイルを使用する際はHTMLに外部ファイルのコードを埋め込みます．
基本的なコード方法はHTMLのものと変わりません．

            const canvas = document.getElementById('canvas');
            const ctx = canvas.getContext('2d');
            ctx.fillStyle = 'blue';
            ctx.fillRect(50, 50, 30, 30);

Java Scriptでのcanvasのメソッド

Java Scriptでcanvasを使用する際のメソッドには以下のようなものがあります．

基礎的なメソッド

  • getElementById(canvasid) - HTMLのcanvas要素を取得します．
  • getContext("2d") - 2D描画コンテキストを取得します．
  • fillStyle - 塗りつぶしの色を設定します．
  • strokeStyle - 線の色を設定します．
  • lineWidth - 線の幅を設定します．
  • clearRect(x, y, width, height) - 座標(x, y)に幅width, 高さheightの矩形領域をクリアします．
  • fillRect(x, y, width, height) - 座標(x, y)に幅width, 高さheightの矩形領域を塗りつぶします．
  • strokeRect(x, y, width, height) - 座標(x, y)に幅width, 高さheightの矩形の輪郭を描画します．

直線描画

  • beginPath() - 新しいパスを開始します．
  • moveTo(x, y) - 座標(x, y)にパスの開始点を設定します．
  • lineTo(x, y) - 現在のパスから座標(x, y)まで直線を追加します．
  • stroke() - 輪郭をなぞる方法でパスを描画します．
  • closePath() - パスを閉じます．
  • fill() - パスを塗りつぶします．

円描画

  • arc(x, y, radius, startAngle, endAngle, anticlockwise) - 円弧を描画します．
  • arc(x, y, 20, 0, Math.pi*2) - 座標(x, y)に半径20pxの円を描画します．
  • arc(x, y, 20, 0, Math.pi, True) - 座標(x, y)に半径20pxの半円を0radから時計回りに描画します．
  • arc(x, y, 20, 0, Math.pi, False) - 座標(x, y)に半径20pxの半円を0radから反時計回りに描画します.
  • arcTo(x1, y1, x2, y2, radius) - 2点間を結ぶ円弧を描画します．

テキスト描画

  • fillText(text, x, y[, maxWidth]) - 座標(x, y)にテキストを描画します．任意で最大描画幅を指定できます.
  • strokeText(text, x, y[, maxWidth]) - 座標(x, y)にテキストの輪郭を描画します．任意で最大描画幅を指定できます.
  • font - テキストのフォントを設定します．
  • textAlign - テキストの配置方法を設定します．
  • textBaseline - テキストの基準線を設定します．

移動・回転

  • translate(x, y) - パスの位置を(x, y)だけ移動します．
  • rotate(angle) - パスの向きを指定した角度だけ回転します．

アニメーション

  • requestAnimationFrame(callback) - アニメーションフレームを要求します．
  • setInterval(callback, delay) - delayミリ秒間隔でcallbackを繰り返し実行します．
  • setTimeout(callback, delay) - delayミリ秒後にcallbackを1回だけ実行します．
  • clearInterval(intervalId) - setIntervalで設定した繰り返しを停止します．
  • clearTimeout(timeoutId) - setTimeoutで設定したタイマーを停止します．
  • cancelAnimationFrame(requestId) - requestAnimationFrameで設定したアニメーションを停止します．

canvas内でクリックすると1秒ごとに正方形の色が変化し,もう一度クリックするかマウスが離れるとアニメーションが停止されます．

軌道上を円が動くアニメーション．

canvas内にマウスカーソルが入るアニメーションが開始され,マウスが離れるとアニメーションが停止されます．
canvas内を左クリックで加速, 右クリックで減速できます.

まとめ

canvasはHTML5で導入された描画要素で、JavaScriptを使用してグラフィックを描画することができます．
基本的な図形の描画から、アニメーションや画像の操作まで幅広い用途に利用できます．
本ページではcanvasの一部の基本的な使用方法とメソッドについて説明しました．
